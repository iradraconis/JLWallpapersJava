# JLWallpapers

Ein Programm zur Bearbeitung und Aktualisierung von Wallpapern für den `j-lawyer-client`.

## Funktionen

1. Auswahl eines Wallpaper-Ordners.
2. Auswahl des `j-lawyer-client` Installationsordners.
3. Extrahieren des vorhandenen `j-lawyer-client.jar` zur Bearbeitung.
4. Löschen der alten Wallpapers.
5. Hinzufügen der neuen Wallpapers nach Bearbeitung. (Dunkler und Weichzeichner)
6. Re-Komprimierung in eine `.jar`-Datei und Ersetzung der originalen `j-lawyer-client.jar`.

## Voraussetzungen

Stellen Sie sicher, dass Sie die richtigen Abhängigkeiten in Ihrer Umgebung installiert haben:

- Java Development Kit (JDK)
- Java Runtime Environment (JRE)

## Verwendung

Öffnen Sie ein Terminal im Ordner der Java Datei. Führen Sie das Programm aus, um das Hauptfenster anzuzeigen. 

```
java JLWallpapers.java
```

Folgen Sie dann den Schritten:

1. Klicken Sie auf "Wallpaper Ordner auswählen" und wählen Sie den Ordner, der die neuen Wallpapers enthält.
2. Klicken Sie auf "j-lawyer Client Ordner auswählen" und wählen Sie den Installationsordner von `j-lawyer-client`.
3. Klicken Sie auf "Start", um den Vorgang zu beginnen.

## Hinweis

Bitte stellen Sie sicher, dass Sie eine Sicherungskopie Ihrer `j-lawyer-client.jar` und aller anderen relevanten Dateien haben, bevor Sie dieses Tool verwenden. Das Programm wird die vorhandenen Wallpapers ersetzen und die originale `.jar`-Datei aktualisieren.
